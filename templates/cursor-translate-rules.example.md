# Global cursor-translate rules (optional)

Copy to `~/.cursor/translate-proxy/cursor-translate-rules.md` and edit.

Or add a `## cursor-translate` section to project `CURSOR.md` / `AGENTS.md`,
or create `.cursor/cursor-translate.md` in the repo.

Example project-specific rules:

- Keep product names in Russian when they are customer-facing brands.
- Translate "бэклог" as "backlog", never "reserve list".
- For crypto3: never translate gate task IDs or ROADMAP section anchors.
- Preserve mermaid node labels that are already in English.
